# 🚀 <b>Run Locally!</b>

## Clone the project

```bash
  git clone https://github.com/z7pz/giveaways.git
```

## Go to the project directory

```bash
  cd giveaways
```

## Install dependencies with:

### `npm`

```bash
  npm install
```

### `yarn`

```bash
yarn
```

### `pnpm`

```bash
pnpm install
```

## Then run the project and have fun! 😀

```bash
  npm run start
```
